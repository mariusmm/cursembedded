FreeRTOS I2C Application example with FreeRTOS

This example project shows how to get data from a sensor (APDS-9960) and send it to other task.

This task changes PWM duty cycle (and hence, LED power) proportionally to the sensor data.

This project uses the following libraries:

  • BSP
  • APDS_9960
  • I2C_Wrapper
  • PWM
